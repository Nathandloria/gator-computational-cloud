GCC is a tool that allows for the creation, planning, and execution of budget-deadline-constrained workflows. With the help of this tool, the user is able to upload source code and data required for certain nodes of an execution, as well as a workflow specific xml file which allows the program to know which nodes must be executed, and in what order to do so. The user can then select a budget and time deadline which the tool will use to optimize both the efficiency and cost of the execution. The 'View Workflows' tab at the top of this page will get you started on your workflows so you can start taking advantage of this tool!

Since GCC uses Amazon AWS and Dropbox as a backend, please provide the tool with updated credentials in order for any executions to work properly!
